# 我的实战

1. 基本概念：酷炫、明亮、画面感
2. 产品：人工智能写作辅助工具
3. 产品特点与风格：认知语言学 - 写作古典文风，人工智能技术，自动纠错节省用户时间
4. 用户人群：文字处理工作者（编辑）、自由写作爱好者
5. 用户年龄：20-35
6. 带来什么样的惊喜：高效率，清爽，酷炫
